الرئيسية sql ربط مع vs العلاقات

ماهي قواعد البيانات :

هي مجموعة من المعلومات المرتبطة ببعضها البعض بطريقة منطقية، يتم تخزينها إلكترونيًا في نظام كمبيوتر. تُستخدم قواعد البيانات لتخزين وتنظيم وإدارة كميات كبيرة من البيانات، مما يسهل على المستخدمين الوصول إليها ومعالجتها.

أنواع قواعد البيانات :

1- قواعد البيانات العلائقية (Relational Databases): هي أكثر أنواع قواعد البيانات شيوعًا. تُستخدم قواعد البيانات العلائقية لتخزين البيانات في شكل جداول، حيث تُمثل كل صف سجلًا من البيانات، والأعمدة تُمثل حقول البيانات. تُستخدم العلاقات لربط الجداول ببعضها البعض.

بعض انواع قواعد البيانات العلائقية :

  • MySQL: هي قاعدة بيانات مفتوحة المصدر شائعة الاستخدام.
  • Oracle: هي قاعدة بيانات تجارية قوية.
  • Microsoft SQL Server: هي قاعدة بيانات تجارية شائعة الاستخدام.

2- قواعد البيانات غير العلائقية (NoSQL Databases): هي أنواع جديدة نسبيًا من قواعد البيانات تُستخدم لتخزين أنواع مختلفة من البيانات، مثل البيانات غير المهيكلة والبيانات المتدفقة.

بعض انواع قواعد البيانات اللاعلائقية

  • MongoDB: هي قاعدة بيانات مستندية شائعة الاستخدام.
  • Cassandra: هي قاعدة بيانات مفتوحة المصدر موثوقة.
  • Redis: هي قاعدة بيانات ذاكرة تخزين مؤقت شائعة الاستخدام.
  • HBase: هي قاعدة بيانات مفتوحة المصدر تُستخدم لتخزين البيانات الكبيرة.

    تُستخدم قواعد البيانات غير العلائقية في مجموعة متنوعة من التطبيقات، بما في ذلك:

  • التطبيقات الكبيرة (Big Data Applications): تُستخدم قواعد البيانات غير العلائقية لتخزين وتحليل كميات كبيرة من البيانات.
  • التطبيقات المتدفقة (Streaming Applications): تُستخدم قواعد البيانات غير العلائقية لتخزين البيانات المتدفقة، مثل البيانات من أجهزة الاستشعار.
  • التطبيقات الاجتماعية (Social Applications): تُستخدم قواعد البيانات غير العلائقية لتخزين البيانات الاجتماعية، مثل بيانات المستخدمين من مواقع التواصل الاجتماعي.

مالفرق بين ال SQL و ال MYSQL

الفرق الرئيسي بين SQL و MySQL هو أن SQL لغة استعلام بينما MySQL برنامج.

SQL (Structured Query Language) هي لغة استعلام تُستخدم للتفاعل مع قواعد البيانات العلائقية. تُستخدم SQL لإنشاء قواعد البيانات وإدارتها، بالإضافة إلى استعلام البيانات وتحديثها وحذفها.

تستخدم SQL للتفاعل مع قواعد البيانات العلائقية،

MySQL هي قاعدة بيانات مفتوحة المصدر وعلائقية تستخدم SQL. تُعد MySQL واحدة من أكثر قواعد البيانات شيوعًا في العالم، وتُستخدم في مجموعة متنوعة من التطبيقات، بما في ذلك مواقع الويب والمنتديات والتطبيقات التجارية.

MySQL قاعدة بيانات علائقية تستخدم SQL.

ماهي اوامر ال DDL :

DDL هو اختصار لـ Data Definition Language، وهو مجموعة من أوامر SQL التي تُستخدم لإنشاء وإدارة بنية قاعدة البيانات. تتضمن أوامر DDL إنشاء قواعد البيانات والجداول والعناصر الأخرى في قاعدة البيانات، بالإضافة إلى تعديلها وحذفها. فيما يلي بعض الأوامر الشائعة لـ DDL :

  • CREATE DATABASE: يُستخدم هذا الأمر لإنشاء قاعدة بيانات جديدة.
  • CREATE TABLE: يُستخدم هذا الأمر لإنشاء جدول جديد.
  • ALTER TABLE: يُستخدم هذا الأمر لتعديل جدول موجود.
  • DROP DATABASE: يُستخدم هذا الأمر لحذف قاعدة بيانات موجودة.
  • DROP TABLE: يُستخدم هذا الأمر لحذف جدول موجود.
  • TRUNCATE: تستخدم لحذف جميع البيانات من جدول دون حذف الجدول نفسه.
  • RENAME: تستخدم لتغيير اسم كائن موجود في قاعدة البيانات.

ماهي اوامر ال DML :

DML هي اختصار لـ Data Manipulation Language، وهي لغة استعلام تستخدم لإضافة وحذف وتعديل البيانات في قاعدة البيانات. DML هي جزء من لغة SQL، وهي اللغة القياسية لقواعد البيانات العلائقية. فيما يلي بعض الأوامر الشائعة لـ DML :

  • INSERT: تستخدم لإضافة بيانات جديدة إلى جدول.
  • SELECT: استرداد البيانات من جدول بناءً على شرط.او استرداد جميع البيانات.
  • UPDATE: تستخدم لتعديل بيانات في جدول.
  • DELETE: تستخدم لحذف بيانات من جدول.
